### Frontend Stack
- **HTML5** (semantic, accessible)
- **Tailwind CSS** (utility-first, responsive)
- **Vanilla JavaScript** (no dependencies)
- **Google Fonts** (Space Grotesk, Inter, JetBrains Mono)

### Design System (from AGENT.md)
- **Colors**: Neural Spectrum gradient (5 colors)
- **Fonts**: 3 custom fonts (headings, body, code)
- **Components**: Cards, buttons, progress bars, grids
- **Breakpoints**: Mobile (640px), Tablet (1024px), Desktop (1280px+)

### Responsive Design
- Mobile-first approach
- Touch targets: 44px minimum
- Font scaling with clamp()
- Flexible grids and layouts

### Performance
- < 50KB per page (minified)
- LCP < 1s
- No JavaScript frameworks
- GPU-accelerated animations

## üéØ Quick Start for Developers

### Adding a New Level (Future)
1. Copy `promptforge-l6.html`
2. Update:
   - Level title & number
   - Color gradient
   - Quest descriptions & XP
   - Mini-boss challenge
   - Final boss requirements
   - Skill tree unlocks
3. Create spec document
4. Test at 3 breakpoints
5. Update game hub links

### Modifying a Level
1. Open the HTML file (e.g., `promptforge-l3.html`)
2. Update:
   - Hero banner gradient (line ~55)
   - Quest content (line ~240+)
   - Mini-boss details (line ~300+)
   - Final boss rewards (line ~400+)
3. Format with `format_file` tool
4. Test responsive design
5. Verify color contrast

### Common CSS Changes
- **Colors**: Update `--neural-spectrum-*` in line 24-29
- **Fonts**: Already imported from Google Fonts
- **Spacing**: Use Tailwind classes (px-4, py-6, gap-4)
- **Animations**: Update `@keyframes` in `<style>` section

## üìû Support & FAQ

### Q: How do I update a level?
A: Edit the corresponding HTML file and update quest descriptions, XP rewards, boss requirements, etc.

### Q: How do I add a new quest?
A: Copy a quest item (`<div class="quest-item">`), update the title, description, and XP reward.

### Q: How do I change colors?
A: Edit the hero banner gradient (line ~55) and the border-left colors in quest items (line ~60).

### Q: How do I add mobile support?
A: All pages already use Tailwind responsive classes. Just test at 640px, 1024px, and 1280px.

### Q: Where are skill trees managed?
A: Skill tree unlocks are listed in each level's final boss "UNLOCK ON DEFEAT" section.

### Q: How do I connect to the backend?
A: Wait for Phase 2 backend API. Then update button click handlers in JavaScript section (line ~435+).
