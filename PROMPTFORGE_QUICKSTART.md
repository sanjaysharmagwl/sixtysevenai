### 1. Build Real Systems
Don't just read guides. Actually:
- Write prompts in ChatGPT or Claude
- Test them with real inputs
- Document your results
- Submit actual outputs as proof

### 2. Test Before Submitting
- Does your prompt produce consistent results?
- Does output match the criteria?
- Can you explain how it works?
- Have you tested edge cases?

### 3. Progress Linearly
- Don't skip quests

### 4. Document Your Work

### 5. Use Power-Ups
- Activate **Focus Mode** for deep work (25 min)
- Use **Memory Lock** to retain concepts
- Grab **XP Multiplier** before big pushes
- Stack multipliers for maximum gain
